# 校园网自动登录工具

## 项目简介

校园网自动登录工具是一个专为校园网络环境设计的自动化登录软件，支持深圳大学等使用类似认证系统的校园网络。该工具可以自动检测网络状态、连接WiFi和有线网络并完成认证登录，同时支持开机自启和后台运行。

## 功能特性

- **自动登录**：自动检测网络状态并完成校园网认证登录
- **多服务器支持**：内置主服务器和备用服务器，提高登录成功率
- **WiFi连接管理**：自动连接到目标WiFi网络（SZU_CTC&CMCC）
- **有线网络支持**：自动检测并认证有线网络连接
- **双网络模式**：支持WiFi和有线网络同时连接
- **开机自启**：支持设置开机自动运行和登录
- **后台运行**：支持最小化到系统托盘，减少对桌面的占用
- **单实例运行**：防止多个实例同时运行
- **日志记录**：详细的运行日志，便于问题排查
- **密码加密**：支持密码加密存储，提高安全性
- **网络监控**：后台监控网络状态，断线自动重连

## 系统要求

- Windows操作系统
- Python 3.7或更高版本
- 网络连接权限

## 安装依赖

项目依赖以下Python库：

```
requests
PyQt5
pystray
PIL (Pillow)
cryptography
pywin32
```

安装依赖：
```bash
pip install -r requirements.txt
```

## 使用方法

### 直接运行Python脚本

1. 确保已安装所有依赖
2. 运行主程序：
   ```bash
   python 校园网登录.py
   ```

### 运行打包的可执行文件

项目根目录下提供了打包好的可执行文件：
```
dist/校园网自动登录.exe
```

双击运行即可。

## 开发说明

本项目采用模块化设计，主要包含以下模块：

- 网络请求模块：处理校园网认证请求
- WiFi管理模块：检测和连接WiFi网络
- GUI界面模块：使用PyQt5构建用户界面
- 系统集成模块：开机自启、托盘图标等功能
- 配置管理模块：处理用户设置和账号信息
